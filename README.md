# 🚨 Crime Rate Dashboard — Excel Data Analytics Project

![Dashboard Preview](dashboard_screenshot.png)

## 📌 Project Overview

This project is an **interactive Crime Rate Analysis Dashboard** built entirely in **Microsoft Excel**. It visualizes law enforcement data across different police units and ranges in Bangladesh, enabling quick comparison of crime categories by region.

---

## 📊 Dashboard Features

| Chart | Description |
|---|---|
| **Dacoity vs Robbery** | Unit-wise grouped bar chart comparing dacoity and robbery counts |
| **Murder vs Kidnapping** | Clustered bar chart showing murder and kidnapping across ranges |
| **Speedy Trial vs Riot** | Horizontal stacked bar chart with % breakdown by unit |
| **Burglary vs Theft** | Line chart tracking trends across DMP, Dhaka, Chittagong, Khulna & Barisal |
| **Arms Act vs Smuggling** | Horizontal bar chart comparing arms and smuggling cases by range |

---

## 🔍 Key Insights

- **Dhaka Range** records the highest murder count (854 cases)
- **Rajshahi Range** leads in smuggling cases (1,538)
- **DMP** shows the highest speedy trial completion rate (216 cases)
- **Khulna Range** has notable burglary figures (683)
- **Chittagong Range** reports significant dacoity activity (96 cases)
